InflamEase is a cross-platform mobile application designed to help individuals **track, understand, and manage chronic inflammation** through informed dietary and lifestyle choices.  
The app acts as a personal health companion, combining food logging, flare tracking, insights, and curated anti-inflammatory content into one intuitive experience.

ðŸ“± Project Vision

The goal of **InflamEase** is to empower users living with chronic inflammatory conditions by providing:

- Awareness of dietary and lifestyle triggers  
- Tools to track flare-ups and symptoms  
- Data-driven insights into inflammation patterns  
- Access to anti-inflammatory recipes and wellness guidance  

The application is built using React Native with Expo, enabling support for both **Android and iOS** platforms.
